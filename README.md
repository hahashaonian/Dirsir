# 网上电子商城

#### 介绍
第一个和同学一起协作完成的较为完整的项目，用的是Java EE原生技术开发，其中包括JSP，Servlet等。

 **软件架构** 
主要使用三层架构
数据访问层：DAO设计模式，JDBC，MySQL
业务层：Service
表现层：JSP、Servlet

开发环境：Eclipse Web服务器：Tomcat 数据库：MySQL
